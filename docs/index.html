§ 03 — transformar dados

Cruzar bases, ler conexões.

As bases do audiovisual brasileiro não falam entre si por acidente — elas se ligam por chaves definidas: o CPB, o NUMERO_SALIC, o CNPJ e, quando nenhum desses está disponível, o título normalizado como proxy. Escolha um cruzamento pronto ou leia as regras antes de montar o seu próprio.

CPB — Código de Produto Brasileiro

O CPB é o identificador único emitido pela ANCINE no momento do registro de cada obra audiovisual. É o elo mais confiável entre as bases — uma obra identificada pelo CPB pode ser ligada sem ambiguidade a seus diretores, produtoras e dados de bilheteria.

Formato: B[YY][seq. 7 dígitos][00000] para obras brasileiras e E[...] para obras estrangeiras (ROE).

Presente em
obras-nao-publicitarias → campo CPB
diretores-obras → campo CPB
produtores-obras → campo CPB
bilheteria-agregada → campo CPB_ROE (prefixo B = obra brasileira, E = estrangeira)
Join direto, sem normalização. Cobertura desde 2002 com CPB retroativo para obras mais antigas.
Join aproximado
Proxy título — normalização por nome

Quando o CPB não está disponível na base de destino (como FSA, bilheteria consolidada ou Lumiere), o elo é feito por normalização do TITULO_ORIGINAL da obra contra o TITULO_PROJETO da base alvo.

title.toLowerCase()
  .normalize('NFD').replace(/[̀-ͯ]/g, '')
  .replace(/[^a-z0-9 ]/g, '').trim()
Usado em
obras → fsa-projetos (TITULO_PROJETO)
obras → bilh-consolidada (título sem CPB)
obras → lumiere-europa (título internacional)
bilh-agregada.TITULO_BRASIL → lumiere (cruzado)
Risco de falso positivo em títulos curtos (“Sol”, “Amor”, “Brasil”). Sempre validar manualmente uma amostra do join antes de usar em análise.
Via sistema MinC
CPB ↔ SALIC — ligação com a Lei do Audiovisual

A base de renúncia fiscal exportada pela ANCINE contém o NUMERO_SALIC, o identificador do projeto no Sistema de Apoio às Leis de Incentivo à Cultura (MinC). Não há campo CPB nessa base.

O elo CPB↔SALIC exige uma das duas estratégias:

Estratégia 1 — proxy título
normalizar(obras.TITULO_ORIGINAL) = normalizar(renuncia.TITULO_PROJETO)
risco: homônimos e variações de título
Estratégia 2 — CNPJ (empresa-level)
obras.CNPJ_REQUERENTE = renuncia.CNPJ_PROPONENTE
join N:M — empresa com múltiplos projetos
Quando o dataset SALIC/MinC for incorporado ao catálogo, o elo direto CPB↔SALIC poderá ser feito via campo de referência cruzada. O NUMERO_SALIC (ex: 00-0040) é diferente do CPB — não confundir.
Janela de distribuição
CRT — Certificado de Registro de Título

O CRT não é um identificador de obra — é um certificado condicionado ao pagamento de CONDECINE (Contribuição para o Desenvolvimento da Indústria Cinematográfica Nacional). Sem CRT, uma obra não pode ser veiculada comercialmente no Brasil.

O CRT é emitido por janela de distribuição: uma obra pode ter um CRT para salas de cinema, outro para TV por assinatura e outro para streaming (VoD). Cada janela exige um certificado separado vinculado ao pagamento do tributo correspondente.

O que o CRT indica
CRT para salas de cinema → obra foi comercialmente exibida em cinemas
CRT para TV por assinatura → obra foi veiculada em TV paga
CRT para VoD / streaming → obra foi disponibilizada em plataforma
Indicador de circulação, não prova de audiência
Ligação com obras
CRT referencia a obra via CPB → join exato (quando o dataset CRT for incorporado)
Enquanto ausente do catálogo: obras.SEGMENTO_DESTINACAO_INICIAL registra a janela declarada no ato do registro — proxy imperfeito, pois é a intenção, não o CRT efetivo
Dataset CRT é uma lacuna conhecida do catálogo. Quando incorporado, permitirá mapear quais obras efetivamente circularam em cada janela — informação de alto valor para análises de mercado e política pública.
