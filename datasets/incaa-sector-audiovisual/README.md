# INCAA / Argentina — Sector Audiovisual (2001–2023)

Dados do setor audiovisual argentino: espectadores e receita por origem do filme, estrenos por origem, dados por provincia, participação de distribuidoras, postos de trabalho em longas-metragens e dados de TV por assinatura. Fontes: INCAA, SICA, ENACOM.
